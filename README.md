# tekkie.me

Online CV of Georgiana Gligor.

## Development Environment

Ensure grunt is properly set up. This only needs to be done once.
```bash
$ npm install -g grunt-cli
$ cd source/themes/tekkie/cv
$ npm install
```

Build the stylesheet:
```bash
$ cd source/themes/tekkie/cv
$ grunt
```
